  actual decision process.

  Primary use is qualitative — we want to detect whether participants
  are conflating "this summary is accurate / complete / fair" with "I
  agree with my partner's underlying view." The phrasing is
  deliberately neutral so we don't telegraph the concern; participants
  who spontaneously talk about agreement vs. accuracy give us the
  signal. Worded so bullets, sentences, or short paragraphs all work.
